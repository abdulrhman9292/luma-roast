# LUMA ROAST

A premium cinematic specialty coffee website built with Next.js 16, TypeScript, Tailwind CSS, GSAP ScrollTrigger, Framer Motion, and Lenis.

## Scripts

```bash
npm.cmd run dev
npm.cmd run build
npm.cmd run lint
```
